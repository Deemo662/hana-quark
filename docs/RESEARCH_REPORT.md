# A股量化选股系统 —— 全局研究报告

> 生成日期: 2026-07-01
> 基于: 董鹏飞《基本面量化投资策略》(2022) + 2022-2026市场变化调研 + 个人中线波段需求

## 一、原方案回顾

### 1.1 书本策略概要

| 要素 | 书本设定 |
|------|----------|
| 回测区间 | 1998/12/31 ~ 2020/12/31（22年，3轮牛熊） |
| 基准收益 | 整体股票等权年化 10.0%~10.5% |
| 股票池 | 全A股 - 市值后15% - ST - 停牌 - 上市<180天 |
| 调仓频率 | 月度调仓，年换手100% |
| 仓位分配 | 等权重，20只或40只 |

### 1.2 书中TOP 5多因子模型

| 排名 | 因子组合 | 22年年化 |
|------|----------|----------|
| 1 | 市值 + 毛利率 + ROIC + 6月波动率 + PS | 18.44% |
| 2 | EV/Sales + ROE + 权益/带息债务 + 6月波动率 + 3月动量 | 17.26% |
| 3 | ROE + 6月波动率 | 16.90% |
| 4 | 中小市值改进神奇公式 | 16.48% |
| 5 | PB + 6月波动率 | 15.14% |

### 1.3 原方案的三个隐含假设

| 假设 | 成立条件 | 当前状态 |
|------|----------|----------|
| 市场结构稳定，因子回报持续 | 审批制下的壳价值逻辑 | ❌ 注册制后壳价值终结 |
| 基本面因子能穿越牛熊 | 长期持有，不择时 | ❌ A股牛短熊长，满仓扛回撤不现实 |
| 学术研究框架可直接移植 | 因子定义与美股一致 | ❌ A股有独特机制（T+1、涨跌停、抱团文化） |

## 二、市场结构变化：2022-2026

### 2.1 注册制改革

```
时间线:
2019.07  科创板注册制试点
2020.08  创业板注册制
2023.02  主板全面注册制落地

结果:
- 上市公司数量: 4,100(2021) → 5,500+(2025)
- 新增企业特征: 科技/生物医药为主，大量未盈利，轻资产
- 壳价值: 从30-40亿(2016) → 不足5亿(2023)
```

### 2.2 因子有效性变化

| 因子 | 1998-2020（书本） | 2022-2026（当前） | 原因 |
|------|-------------------|-------------------|------|
| **市值（小盘）** | ⭐⭐⭐⭐⭐ 最强 | ⭐⭐ 明显衰减 | 壳价值消失，小盘不再自带溢价 |
| **PE/PB（低估值）** | ⭐⭐⭐⭐ 有效 | ⭐⭐⭐ 需升级 | 未盈利企业PE为负，科技股PB失真 |
| **ROE/ROIC（质量）** | ⭐⭐⭐⭐ 有效 | ⭐⭐⭐⭐⭐ 增强 | 劣质企业更容易上市，质量筛选价值↑ |
| **毛利率** | ⭐⭐⭐⭐ 有效 | ⭐⭐⭐⭐ 稳定 | 上下游定价权是硬道理 |
| **6月波动率（低波）** | ⭐⭐⭐⭐⭐ 极强 | ⭐⭐⭐⭐⭐ 极强 | A股低波动异象持续稳定 |
| **股息率** | ⭐⭐⭐ 中等 | ⭐⭐⭐⭐⭐ 爆发 | 2023-2025红利策略最强 |
| **价格动量** | ⭐⭐ 弱 | ⭐⭐ 仍弱 | A股反转效应>动量效应 |
| **经营现金流/利润** | ⭐⭐⭐ 排雷用 | ⭐⭐⭐⭐ 排雷价值↑ | 注册制后"注水利润"更普遍 |

### 2.3 资金抱团周期

A股四轮抱团历史：

```
2007-2009  金融地产  →  2013-2015  TMT/创业板  →  2017-2020  核心资产  →  2020-2021  新能源
                                                                                    ↓
2024-2025  AI/半导体  ←  2023-2024  红利低波  ←  2022-2023  抱团全面瓦解
```

判断：抱团是**周期现象**，不是结构特征。每轮约3-5年，经历形成→高潮→瓦解→真空四个阶段。在机构化加速背景下（公募规模从14.8万亿→27.6万亿），抱团强度持续加大。

## 三、策略升级方案

### 3.1 核心原则

1. **去市值因子** —— 壳价值逻辑已终结，小市值不再自带Alpha
2. **加质量排雷** —— 注册制下垃圾股更多，排雷价值大于选牛
3. **加股息率** —— 2023-2025最有效的新因子
4. **紧股票池** —— 不能用简单的"后15%剔除"，需要多维过滤
5. **分阶段应对抱团** —— 抱团周期不同阶段用不同策略

### 3.2 三层策略架构

```
┌─────────────────────────────────────────────┐
│              📊 节奏层（择时）                │
│  抱团阶段监测 → 仓位动态调整                  │
│  高潮→半仓 | 瓦解后→加仓 | 真空→满仓          │
└──────────────────┬──────────────────────────┘
                   │
┌──────────────────┴──────────────────────────┐
│              🔬 增强层（新因子）              │
│  股息率 | 北向资金 | 机构拥挤度 | 分析师预期   │
│  （后三项待接JQData）                        │
└──────────────────┬──────────────────────────┘
                   │
┌──────────────────┴──────────────────────────┐
│              🏛 基础层（质量+低波+价值）      │
│  ROE/ROIC | 毛利率 | 6月波动率 | PB/PS        │
│  经营现金流排雷 | 多维股票池过滤              │
│  ★已移除：市值因子（注册制后失效）            │
└─────────────────────────────────────────────┘
```

### 3.3 新策略定义

#### 策略A：质量低波（主策略，替换原TOP1）

| 参数 | 值 |
|------|-----|
| 因子 | ROE + 毛利率 + 6月波动率 + PB + 经营现金流/利润 |
| 逻辑 | 好公司 + 低波动 + 便宜 + 利润真实 |
| 调仓 | 月度，TOP 20-30只，等权 |
| 适用 | 全市场阶段 |

#### 策略B：红利低波（2023-2025验证最强）

| 参数 | 值 |
|------|-----|
| 因子 | 股息率 + 6月波动率 |
| 逻辑 | 高分红 + 低波动 = 确定性溢价 |
| 调仓 | 季度，TOP 30只 |
| 适用 | 熊市/震荡市 |

#### 策略C：质量红利（均衡型）

| 参数 | 值 |
|------|-----|
| 因子 | ROE + 股息率 + 6月波动率 |
| 逻辑 | 能赚钱 + 愿分红 + 波动小 |
| 调仓 | 月度，TOP 25只 |

### 3.4 股票池升级

| 过滤维度 | 旧规则（书本） | 新规则 |
|----------|---------------|--------|
| 市值 | 剔除后15% | 剔除后20% + 日均成交额>500万 |
| ST/*ST | 剔除 | 剔除 |
| 新股/次新股 | 上市<180天 | 上市<2年（500个交易日） |
| 停牌 | 剔除 | 剔除 |
| PE为负 | 剔除（书中建议） | 剔除 |
| **新增** | — | 分析师覆盖≥2家（待JQData） |
| **新增** | — | 日均换手率>0.1%（防僵尸股） |

### 3.5 抱团阶段监测指标

| 指标 | 数据源 | 阈值 | 动作 |
|------|--------|------|------|
| 行业成交额集中度 | 市场数据 | 前3行业 > 50% | 警惕高潮 |
| 基金发行规模 | 公开数据 | 月度 > 2000亿 | 后期信号 |
| 北向资金方向 | 沪深港通 | 连续流出 > 5日 | 减仓信号 |

## 六、关键风险和注意事项

1. **数据质量**：Baostock的财务数据拉取速度极慢（每只约30秒），全量数据需要分批进行
2. **因子周期性**：任何因子都有逆风期。红利策略可能在牛市中跑输成长策略
3. **过拟合风险**：不要因为2023-2025红利表现好就过度依赖红利因子
4. **实盘偏差**：回测结果≠实盘结果。手续费、滑点、T+1、涨跌停在实盘中影响更大
5. **心理因素**：策略最大回撤可能超过20%，需要心理准备
