# V2 短线策略规划

> 分支: v2-short-term | 日期: 2026-07-01
> 背景: 基于现有 v1 基本面量化系统的架构，扩展短线热点策略模块

## 一、方法论来源评估

不同于 v1 有董鹏飞《基本面量化投资策略》（53单因子+31多因子，A股22年回测）作为单一权威来源，短线领域方法论分散。以下按可靠性排序：

### S级（券商金工研报，有样本外验证）

| 来源 | 策略 | 回测区间 | 年化收益 | 夏普 | 最大回撤 | 可靠性指标 |
|---|---|---|---|---|---|---|
| 华安证券 2026.03 | 首板回调策略 | 2021-2025 | 18.21% | 1.09 | -9.69% | 样本外20.72%，优于样本内 |
| 华安证券 | 情绪周期五阶段仓位管理 | 同上 | — | — | — | 作为仓位调节因子 |

**评估**: 华安首板回调是当前能找到的最可靠的A股短线量化策略来源——有完整的样本外验证、辛普森悖论检验、成本敏感性分析。与沪深300相关性仅0.14，可叠加到现有基本面策略上作为Alpha增强。

### A级（学术论文，有统计显著性）

| 来源 | 发现 | 可量化程度 |
|---|---|---|
| Chu, Gu, Zhou (2019) | A股日内动量：开盘半小时收益率预测尾盘方向 | 高（可提取为因子） |
| Da, Liu, Schaumburg (2013) | 短期反转 vs 流动性供给 | 中 |
| Zeng & Liu | A股短期动量（1月持有）+中期反转（9-12月） | 高 |
| MDPI 2025 | 开盘量能不确定性预测尾盘，XGBoost准确率71% | 高（日内因子） |

### B级（业界实证，需谨慎）

| 来源 | 发现 | 注意 |
|---|---|---|
| 腾讯新闻 2025 | 110,805次涨停统计：涨停次日上涨概率62.66%，但次日买入反亏 | 胜率≠盈亏比 |
| 腾讯新闻 | "烂板"（盘中触板未封）未来正收益 | 与直觉相反，需独立验证 |
| 聚宽策略天梯 | 首板低开策略、涨停回踩策略有多个高收益实现 | 存在过拟合风险 |
| 同花顺回测 | 量比>150%、短线测试等超高收益 | 收益数字可疑，可能是未来函数 |

### D级（不可用）

- 纯主观经验（"涨停敢死队心法"等）：无法量化，无法回测
- 小道消息/内幕：不合法，不可持续

## 二、策略矩阵设计

基于上述评估，设计以下短线策略矩阵，全部可量化、可回测、可跟踪：

### 策略1: 首板回调策略 🔴 S级
**来源**: 华安证券《涨停板背后的Alpha》(2026.03)

```
逻辑: 涨停→次日回调→买入→再涨→卖出
信号: 
  买入: T-1日涨停(非ST/非新股) AND T日开盘涨幅≤3% AND 流通市值<100亿
  卖出: T+1日开盘卖出 (91%交易持1天)
仓位: 情绪周期五阶段模型调节(冰点→重仓, 高潮→轻仓)
```

**书中对照**: 无。这是纯短线逻辑，v1基本面策略完全不涉及。

### 策略2: 量能突破策略 🟠 A级
**来源**: MDPI 2025学术论文 + 业界经验

```
逻辑: 开盘放量+趋势确认→日内动量延续
信号:
  买入: 开盘30分钟量比>2.0 AND 涨幅>3% AND 非涨停
  卖出: 尾盘14:50强制平仓或止损-3%
因子: hot_volume_ratio, hot_open_range_breakout
```

### 策略3: 烂板反转策略 🟠 B级
**来源**: 腾讯新闻2025实证（110,805样本）

```
逻辑: 盘中触涨停但未封住→次日低开→反转买入
信号:
  买入: T-1日盘中触涨停(收盘涨幅>6%且<9.5%) AND T日低开>-3%
  卖出: 持有2天或+5%止盈
因子: hot_broken_board, hot_reversal_gap
```

### 策略4: 龙虎榜主力跟随策略 🟡 B级
**来源**: 东方财富龙虎榜数据

```
逻辑: 机构席位净买入>阈值→跟随→短线持有
信号:
  买入: T日龙虎榜机构净买入>5000万 AND 非一字板
  卖出: 持有3天或+8%止盈/-5%止损
因子: hot_institutional_net_buy, hot_retail_sell_ratio
```

### 策略5: 短线动量策略 🟠 A级
**来源**: Zeng & Liu 学术论文（A股短期动量存在且显著）

```
逻辑: 过去5日涨势最强→短期延续
信号:
  买入: 5日动量前10% AND 换手率>5% AND 非涨停
  卖出: 持有3-5天
因子: hot_momentum_5d, hot_turnover
```

## 三、与v1系统的复用关系

你的理解完全正确：**模拟盘跟踪框架已搭好，多加短线策略成本极低。**

| 模块 | v1已有 | v2需新增 |

| 数据层 | MockProvider | 龙虎榜数据源 、资金流向API |

| 定时任务 | automation每日15:30 | ✅ 复用 |

## 四、验证计划

鉴于你明确说"短线没把握，必须大量模拟盘验证"，设计三级验证：

### 第一阶段: 数据源搭建（1-2天）
- [ ] 接入同花顺/东方财富龙虎榜API
- [ ] 接入涨停板数据（AkShare有 `stock_zt_pool_em`）
- [ ] 接入资金流向数据
- [ ] 扩展Mock数据源生成短线特征数据

### 第二阶段: 因子+策略编码（2-3天）
- [ ] 实现10+短线因子
- [ ] 配置5条短线策略到 strategies_short.yaml
- [ ] 跑通Mock数据端到端

### 第三阶段: 模拟盘验证（2-4周持续）
- [ ] 每天收盘后自动运行全部策略
- [ ] 每周出横向对比评估（5条短线 vs 5条基本面 vs 沪深300）
- [ ] 淘汰胜率<45%或最大回撤>20%的策略
- [ ] 保留的策略进入小资金实盘验证

## 五、Caveat（必须前置的警告）

1. **模拟盘≠实盘**：短线策略最大的敌人是滑点和流动性。华安研报明确指出"实盘中执行质量是决定策略成败的核心变量"。模拟盘能验证信号逻辑，但不能验证成交质量。
2. **容量限制**：首板回调策略年化18%的业绩是以每次几万元的小资金测的，资金量上去收益率会骤降。
3. **市场风格切换**：2021-2025恰好是题材股活跃期。如果未来市场风格转向大盘蓝筹，涨停板策略可能集体失效。
